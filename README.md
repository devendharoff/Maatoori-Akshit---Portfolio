# 🎬 Maatoori Akshith | Cinematic Video Portfolio

![Maatoori Akshith Portfolio Banner]

A high-end, conversion-focused digital experience designed for a premium video editing and motion graphics showcase. This portfolio features a cinematic "studio" light aesthetic, interactive hover-to-play sound logic, and an editorial typography system.

## ✨ Key Features

- **🎯 Cinematic Showcase:** Professional "Best Works" section featuring categorized grids for Motion Graphics and Video Editing.
- **🔊 Hover-Audio Integration:** Instant, zero-lag audio on hover for both local `.mp4` and YouTube Short/Long-form embeds.
- **🎨 Premium Visual Design:** Minimalist studio aesthetic with custom typography (serif vs. sans-serif contrast) and ambient mesh glow effects.
- **📱 Vertical Optimization:** Custom 9:16 aspect ratio grids for short-form content (Instagram Reels, YouTube Shorts).
- **🦾 Powered by AI:** Seamless integration of modern front-end tech including Framer Motion, Tailwind CSS, and the YouTube IFrame API.

## 🛠️ Technology Stack

- **Core:** React 18 with TypeScript
- **Styling:** Tailwind CSS (v4 compatible)
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **API:** YouTube JavaScript API

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Production Build:**
   ```bash
   npm run build
   ```

## 🎥 Content Management

The portfolio content is managed via a centralized `portfolioData` object in [src/App.tsx](src/App.tsx), allowing for easy updates of YouTube IDs and local video files.
